# KLResolute WhatsApp SaaS MVP

FastAPI skeleton for WhatsApp automation SaaS.

## What this is
- Deployable FastAPI app
- Render free-tier compatible
- PostgreSQL connectivity check
- No business logic yet

## Endpoints
- GET / → basic service check
- GET /health → app health
- GET /health/db → database connectivity
